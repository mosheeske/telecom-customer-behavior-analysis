# 📊 Telecom Customer Behavior Analysis

## 📌 Overview
This project analyzes customer behavior in a telecommunications company, focusing on usage patterns across different service plans. The analysis includes data cleaning, aggregation, and comparison of key metrics such as calls, messages, and internet usage.

## 🎯 Objective
To understand how customers use telecom services across different plans and identify differences in behavior that could impact pricing, marketing, and business strategy.

## 🛠️ Tools & Technologies
- Python
- pandas
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Aggregation and grouping

## 📂 Dataset
The dataset includes multiple tables related to telecom users:
- User information
- Call records
- Message activity
- Internet usage
- Plan details

## 🔍 Key Steps
- Loaded and merged multiple datasets into a unified structure
- Cleaned missing values and ensured data consistency
- Calculated monthly usage metrics per user:
  - Number of calls
  - Total call duration
  - Number of messages sent
  - Internet data consumption
- Aggregated user behavior by plan type
- Compared usage patterns between plans
- Analyzed differences in customer activity levels

## 📈 Key Findings
- Significant differences in usage behavior between plans
- Some plans showed higher internet consumption but fewer calls
- Messaging behavior varied across user segments
- Usage patterns suggest opportunities for pricing optimization

## 💡 Business Impact
This analysis can help telecom companies:
- Optimize pricing strategies
- Adjust plan structures based on real usage
- Identify high-value customer segments
- Improve targeting for marketing campaigns
- Reduce churn through better plan alignment
